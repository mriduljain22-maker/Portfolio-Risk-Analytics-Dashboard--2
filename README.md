# Portfolio Risk Analytics Dashboard

This project analyzes the risk and performance of an equity portfolio using quantitative risk metrics and data visualization.

The goal of the project is to simulate how market risk teams in financial institutions monitor portfolio risk using tools such as Python, SQL, and Power BI.

---

## Project Overview

This project builds an end-to-end risk analytics workflow including:

Data processing and portfolio return calculation
Risk metric computation
Monte Carlo portfolio simulation
Database storage using SQL
Interactive visualization using Power BI

## Risk Metrics Implemented

### Value at Risk (VaR)

Measures the maximum expected portfolio loss over a given time horizon at a specified confidence level.

Portfolio VaR: **-1.59%**

---

### Expected Shortfall (ES)

Expected Shortfall measures the average loss that occurs beyond the Value at Risk threshold.

Portfolio Expected Shortfall: **-2.60%**

---

### Sharpe Ratio

Sharpe Ratio measures the risk-adjusted return of the portfolio.

Sharpe Ratio: **0.94**

---

### Rolling Volatility

Rolling volatility was calculated to analyze how portfolio risk changes over time.

---

## Monte Carlo Simulation

Monte Carlo simulation was used to generate multiple potential portfolio price paths in order to analyze uncertainty and future portfolio behavior.
